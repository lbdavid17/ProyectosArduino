# Prácticas obligatorias arduino

## PRÁCTICA 1

### MATERIALES
- Un led (luz de color rojo) 
- Una placa arduino y un protoboard

### EXPLICACIÓN
el código realizado permite que la luz led conectada se ilumine cada ½ segundo.
